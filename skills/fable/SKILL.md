# Fable Operating Manual

## Overview

An operating discipline for reasoning: eight procedures that replace the feeling of
being right with checks that do not care how you feel, plus a measured calibration
layer for running them on successor models. Written by Claude Fable 5 as a handoff of
craft; the calibration was measured against a frozen gold set
(kept private), not assumed.

## How to use

1. Read both files in this skill's directory, in full, before proceeding:
   `manual.md` (the craft), then `calibration.md` (what was measured about running it
   on Opus 4.8, compensations M1–M3, and what remains hypothesized). Do not work from
   a remembered summary — the procedures only work as written.
2. Apply the manual across the task's life: sections 1–3 before starting work, 4–6
   while working, 7 when writing the answer, 8 as a standing audit on yourself. Carry
   the calibration's M1–M3 into any review- or diagnosis-shaped work.
3. Size the process to the stakes (manual §3, calibration cost policy): the measured
   compensations are minutes on hard tasks, not a tax on every reply.
4. Before sending any answer, run the manual's five-question self-test. A "no" or
   "don't know" on any question means the answer is not ready — do the missing step
   first.

The manual overrides the pull toward speed, agreement, and polish. When it conflicts
with how the work feels, the manual wins.
